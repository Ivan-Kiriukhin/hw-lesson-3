# Домашнє завдання - Урок 3
## DOM: елементи, класи, стилі, атрибути

> Детальна інструкція з самого початку (встановлення Node.js, Git, форк, клонування, здача завдання) знаходиться в репозиторії першого домашнього завдання.

Відкрийте файл `solution/solution.js` і напишіть код у функціях.

```bash
npm install     # один раз
npm test        # перевірити
```

Після `git push` тести запустяться автоматично на GitHub (дивіться вкладку **Actions**).

## Перевірка в браузері

У цьому уроці з'явився файл `index.html` - там можна побачити свій код у дії.

1. Якщо ще не встановлено - встановіть розширення **Live Server** у VS Code (Extensions → пошук "Live Server" → Install)
2. Відкрийте `index.html` через **Live Server**
3. У браузері навпроти кожної задачі є кнопка **Запустити**: натискайте і одразу побачите результат на сторінці

Це додатковий спосіб перевірити себе, бо ГОЛОВНА ПЕРЕВІРКА все одно через `npm test`.
